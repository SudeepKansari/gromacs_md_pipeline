
# Molecular Dynamics Pipeline for Protein-Ligand Complexes

Automated **GROMACS** pipeline for **protein-ligand MD simulations** using **AmberTools** system preparation (FF19SB + GAFF2).

---

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/SudeepKansari/gromacs_md_pipeline.git
```
**This will be used as volume-mount inside the container; keep the input complex.pdb file here.**


### 2. Build the Docker Image

```bash
docker build -t gromacs .
```

### 3. Run Container (Single-GPU setup)

```bash
docker run -it --rm --gpus all --cpuset-cpus="0-3" -e OMP_NUM_THREADS=4 \
  -v .:/home/dockeruser/workspace gromacs:latest
```

**Or with image ID:**

```bash
docker run -it --rm --gpus all --cpuset-cpus="0-3" -e OMP_NUM_THREADS=4 \
  -v .:/home/dockeruser/workspace <image_id>
```

**Or in a Multi-GPU setup:**

```bash
docker run -it --rm --gpus device=<n> --cpuset-cpus="0-3" -e OMP_NUM_THREADS=4 \
  -v .:/home/dockeruser/workspace <image_id>
```

### 4. Configure `config.json`

Edit the configuration file with your simulation parameters:

| Parameter | Description | Example |
|-----------|-------------|---------|
| `protein_pdb` | Docked **protein-ligand PDB** | `"complex.pdb"` |
| `ligand_resname` | **Ligand residue name** (from HETATM col-4 in PDB) | `"LIG"` |
| `box_type` | Simulation box type: `"Box"` (cubic) or `"Oct"` (truncated octahedron) | `"Box"` |
| `box_distance` | Buffer distance in **Ångströms** from protein | `10.0` |
| `temperature` | Temperature in **K** | `300` |
| `pressure` | Pressure in **bar** | `1.0` |
| `nvt_steps` | NVT equilibration steps | `50000` |
| `npt_steps` | NPT equilibration steps | `50000` |
| `md_steps` | Production MD steps | `50000` |

### 5. Run Pipeline

```bash
python main.py
```

### 6. Output Files

All results are stored in `./output` directory with the following structure:

#### `output/amber/`
Topology and coordinate files converted from AMBER to GROMACS format

#### `output/gromacs/`
GROMACS simulation files:
- MDP parameter files
- GRO structure files
- XTC trajectory files

#### `output/protein_ligand_prepared/`
Prepared protein and ligand files (generated by AmberTools)

#### `output/analysis/`

- **MD_Analysis_Report.pdf** - Comprehensive analysis report including:
  - Protein-Ligand RMSD
  - Protein RMSF (per-residue)
  - Ligand RMSF (per molecule)
  - Radius of Gyration
  - Solvent Accessible Surface Area (SASA)
  - Hydrogen bonding analysis

- **sid_occupancy.png** - Protein-ligand contact map showing:
  - Interaction types
  - Contact duration (%) over the simulation

- **sid_heatmap.png** - Residue-ligand interaction heatmap:
  - Persistent bands indicate stable contacts
  - Color intensity reflects dynamic interaction strength

## Requirements

- **Docker** with GPU support
- **NVIDIA GPU** (recommended)
- AmberTools and GROMACS (included in Docker image)

## Notes

- Ensure your input PDB file is properly formatted with correct HETATM records
- Check PDB file column 4 for correct ligand residue name
- Adjust box distance and MD steps based on your simulation needs
- GPU support requires `nvidia-docker` or Docker with `--gpus` flag
